# Nur 新闻网页爬虫

使用 crawl4ai 库爬取 nur.cn 新闻网站数据的工具。

## 文件说明

- `sc.py` - 主程序脚本，用于爬取 nur.cn 新闻网页
- `nur新闻网页爬取数据.zip` - 爬取的数据文件

## 使用方法

```bash
python sc.py
```

## 数据说明

- **数据数量**: 约 27,407 条新闻文章
- **内容**: 维吾尔语新闻文章，涵盖健康、科技、社会、经济等多个领域
- **文件格式**: .txt 文本文件
- **来源**: nur.cn 网站

## 依赖

- crawl4ai
- beautifulsoup4
- lxml
